# MC2IT Theme
The [Bootstrap](https://getbootstrap.com) theme used by [MC2IT](https://www.mc2it.com) applications.

## Quick start
Install the latest version of **MC2IT Theme** with a package manager:

<!-- tabs:start -->

#### **Haxe**

```shell
lix +lib mc2it_theme
```

#### **JavaScript**

```shell
npm install @mc2it/theme
```
